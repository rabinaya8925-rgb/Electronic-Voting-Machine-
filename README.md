# Electronic-Voting-Machine-
This project implements an Electronic Voting Machine (EVM) using Arduino. It provides a simple and secure voting system where users can cast votes for multiple candidates through push buttons. The system accurately counts votes and displays the results on an LCD screen.

📌 Features

Supports multiple candidates
Accurate vote counting
Real-time result display on LCD
User-friendly interface
Low-cost and easy to implement
Suitable for academic and embedded system projects

🛠 Components Required

Arduino Uno
Push Buttons
16×2 LCD Display
Breadboard
Jumper Wires
Power Supply

⚙️ Working Principle

The voter selects a candidate by pressing the corresponding button.
The Arduino registers and counts the vote.
The LCD displays the voting status.
After the voting process is completed, the total votes for each candidate are displayed.

🚀 Applications

School and college elections
Demonstration of electronic voting systems
Embedded systems learning
Engineering mini and final-year project

🔮 Future Enhancements

RFID-based voter authentication
